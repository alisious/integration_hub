# System Overview

IntegrationHub is designed as a modular integration platform.

## Layers

### Core
- Domain
- Application
- Infrastructure
- API

### Sources
Adapters to external systems:
- SRP
- CEP
- KSIP
- ANPRS
- ZW

### Clients
Client-specific modules (e.g. PIESP)

### Shared
Reusable libraries:
- CSV
- Excel
- Horkos extensions

### Tools
Standalone tools (e.g. HefajstosSync)
